本分类下的网络自媒体指代的是 **视频内容及包装已经形成类似于一般电视台的体系，且相对有组织性**（即便是个人运营）的网络自媒体。

值得注意的是，一些“架空放送局”会以网络自媒体的形式活动。

::: warning
含有黑子性质的网络自媒体不会被收录到此分类中，而是会转到 [放送圈黑子](/web-fair/broadcasting-weirdo/README.md) 分类。
:::
